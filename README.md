# 🏠 Home Automation System with Arduino

A home automation project that allows users to control lights and doors through a mobile application, integrating hardware and software using wireless communication.

---

## 🚀 Overview

This project was developed as part of a technology course with the goal of creating a functional smart home system. It demonstrates the integration between a mobile app and physical components using Arduino.

---

## ⚙️ Features

- 💡 Remote control of lights  
- 🚪 Open and close doors via mobile app  
- 📱 Custom-built mobile interface  
- 📡 Wireless communication (Bluetooth/Wi-Fi)  

---

## 🛠️ Technologies Used

- **Arduino** – hardware control  
- **PictoBlox** – block-based programming  
- **Kodular** – mobile app development  
- **Wireless Communication** – Bluetooth / Wi-Fi  

---

## 📱 How It Works

The mobile application sends commands wirelessly to the Arduino.  
The Arduino processes these commands and controls physical components such as lighting systems and door mechanisms.

---
